Viemo playlist jquery and php Api html5 installation for wordpress

Requires PHP 5 and Curl or url fopen. It is coded using PHP 5 OOP and wordpress 3.x

Unzip the file and upload the dir vimeoplaylist to
wordpress/wp-content/plugins/

or in your admin add new plugin and upoad the zip (recommended)

Go to your admin page and click on plugins - Vimeo SEO Playlist and click activate

Now go to Settings and click on Vimeo Playlist
Here you will see the shortcodes to use for your pages and posts and all kind of settings.

For all the settings examples go to http://www.cfcms.nl/vimeo/

Good luck and happy viewing
